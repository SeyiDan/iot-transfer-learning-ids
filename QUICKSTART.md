## ğŸ› Troubleshooting

### Issue: "Kaggle API credentials not found"
**Solution:** Make sure you entered credentials correctly in the notebook

### Issue: "Google Drive mount failed"
**Solution:** Click "Authorize" button when prompted in Colab

### Issue: "Out of memory error"
**Solution:** Reduce `sample_frac` in preprocessing:
```python
preprocess_dataset(csv_path, dataset_name, sample_frac=0.2)  # Instead of 0.3
```

### Issue: "Module not found"
**Solution:** Install missing package:
```bash
pip install <package_name>
```
